# data-cleaning-visualization-project
## Student Performance Analysis

## Project Overview
This project focuses on cleaning, processing, and visualizing a raw student performance dataset using Python.

The main objective is to identify patterns, trends, and insights from student exam performance data.

---

## Dataset
The dataset contains information about:

- Gender
- Race/Ethnicity
- Parental Level of Education
- Lunch Type
- Test Preparation Course
- Math Score
- Reading Score
- Writing Score

---

## Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Project Workflow

### 1. Data Import
Imported dataset using Pandas.

### 2. Data Cleaning
Performed:
- Missing value handling
- Duplicate removal
- Data type checking
- Outlier detection

### 3. Exploratory Data Analysis
Analyzed:
- Score distributions
- Gender-wise performance
- Correlation between subjects

### 4. Data Visualization
Created:
- Histograms
- Bar Charts
- Boxplots
- Heatmaps

---

## Key Insights

- Reading and writing scores show strong correlation.
- Student performance varies based on preparation course completion.
- Some outliers were identified in math scores.
